# Additional Info

## Insights

- USE RESISTORS (google Pull-down/Pull-up)
- Theres a difference between which side you connect the LED
- Good idea would be to invest in LED's with integrated resistors.
- Get a good power supply and some good quality breadboards.
- You need more wires than you think.


## Tools Used

- Multimeter
- Wire stripper / cutter
- Arduino UNO R4 WiFi
